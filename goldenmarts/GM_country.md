## Глоссарий (общий для всех уровней)

| Термин | Определение |
|---|---|
| ВВП | Валовой внутренний продукт. Сумма стоимости всех товаров и услуг, произведённых на территории страны за год. |
| ВРП | Валовой региональный продукт. То же, что ВВП, но в разрезе одной области. |
| ВТП | Валовой территориальный продукт. То же, что ВВП, но в разрезе одного города или тумана. |
| Возрастная структура | Распределение населения по группам: 0–14, 15–64, 65+ лет. |
| Дефицит бюджета | Превышение расходов над доходами. Профицит – обратная ситуация. |
| Семья | Лица, связанные родством или браком. Могут жить раздельно. |
| Домохозяйство | Лица, живущие в одном жилье и ведущие общий быт. Один человек – тоже домохозяйство. |
| Активный субъект | Юр. лицо или ИП, ведущий деятельность и сдающий отчётность. |
| МСБ | Малый и средний бизнес по узбекскому законодательству. |
| ЯТТ | Якка тартибдаги тадбиркор – индивидуальный предприниматель. |
| Предприятие-экспортёр | Субъект, хотя бы раз осуществивший экспорт в отчётном году. |
| Экономически активное население | Лица 15+ лет, работающие или ищущие работу. |
| Официально занятые | С трудовым договором или зарегистрированные ИП/самозанятые. |
| Неофициально занятые | Работающие без официальной регистрации (оценка Госкомстата). |
| Социальный реестр | Единая база нуждающихся семей, получающих гос. помощь. |
| Эмиграция / Иммиграция | Выезд / прибытие на ПМЖ за год. |
| Младенческая смертность | Число смертей детей до 1 года на 1000 родившихся живыми (‰). |
| Кредитный портфель | Сумма выданных и непогашенных кредитов на конец года. |
| Ипотечные / ИЖС | Кредиты на жильё и индивидуальное жилищное строительство. |
| NPL | Non-Performing Loans – кредиты с просрочкой более 90 дней. |
| Депозитный портфель | Сумма всех вкладов физ. и юр. лиц на конец года. |
| Жилой фонд | Общая площадь всех жилых помещений. |
| Введено жилья | Площадь жилья, введённого в эксплуатацию по актам ввода. |
| Аварийное жильё | Официально признанное непригодным для проживания. |
| Доля аварийного жилья, % | Аварийное жильё ÷ жилой фонд × 100. Считается автоматически. |
| Охват электр./водопр./канализ./газ, % | Подключённые домохозяйства ÷ всего домохозяйств × 100. Считается автоматически. |
| Доля дорог в хорошем состоянии, % | Дороги в хорошем состоянии ÷ все дороги × 100. Считается автоматически. |
| Доля асфальтированных дорог, % | Асфальтированные дороги ÷ все дороги × 100. Считается автоматически. |
| Покрытие 4G | Процент территории с сетью 4G LTE. |
| Ночёвки | Человеко-ночи в объектах размещения. 1 турист × 3 ночи = 3 ночёвки. |
| МТМ | Мактабгача таълим муассасаси – детский сад. |
| Махалля | Орган самоуправления и территориальная единица в Узбекистане. |
| ТБО | Твёрдые бытовые отходы. |
| План 2026 | Целевое значение показателя на конец 2026 года. |
| н/д | Нет данных. Лучше этого, чем 0. |
