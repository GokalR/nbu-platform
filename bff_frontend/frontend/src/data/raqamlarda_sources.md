## Population caption types

The infographic uses two different caption styles for the population number:

- **`asOfDate`** — "По состоянию на DD.MM.YYYY" / "YYYY-yil DD-MMM holatiga"
  Used by: Uzbekistan, Fergana, Samarqand, Qashqadaryo, Xorazm, Namangan,
  Surxondaryo, Sirdaryo, Tashkent city, Karakalpakstan
- **`currentCount`** — "Aholining joriy hisobi boʻyicha" /
  "По текущему учёту населения"
  Used by: Bukhara, Jizzax, Tashkent region
